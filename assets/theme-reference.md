## CSS Usage in Prototypes

Include Inter font in all prototypes:

```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500&display=swap" rel="stylesheet">
```

Use CSS custom properties for theme tokens. Example:

```css
:root {
  /* Primary */
  --primary-dark: #2A1731;
  --primary-darker: #3B1F46;
  --primary-base: #5D2F70;
  --secondary-base: #FDDC8D;

  /* Background (light mode) */
  --bg-strong: #171717;
  --bg-surface: #262626;
  --bg-soft: #EBEBEB;
  --bg-weak: #F7F7F7;
  --bg-white: #FFFFFF;

  /* Text (light mode) */
  --text-strong: #171717;
  --text-sub: #5C5C5C;
  --text-soft: #A3A3A3;
  --text-disabled: #D1D1D1;

  /* Stroke (light mode) */
  --stroke-strong: #171717;
  --stroke-sub: #D1D1D1;
  --stroke-soft: #EBEBEB;

  /* State */
  --success-base: #1FC16B;
  --warning-base: #FA7319;
  --error-base: #FB3748;
  --info-base: #5D2F70;

  /* Font */
  --font-family: 'Inter', sans-serif;
}

body {
  font-family: var(--font-family);
  color: var(--text-strong);
  background: var(--bg-white);
}
```
